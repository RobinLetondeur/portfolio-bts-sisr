Création d'un Active Directory sous Windows Server 2022

Janvier 2023 Infrastructure réseau

Description du projet

Ce projet consistait à mettre en place un Active Directory complet sous Windows Server 2022 pour une entreprise fictive de 50 employés. L'objectif était de centraliser la gestion des utilisateurs, des ordinateurs et des politiques de sécurité.

Fonctionnalités implémentées

  • Création d'un domaine Active Directory avec DNS intégré
  • Configuration des Unités Organisationnelles (OU) par services
  • Mise en place des stratégies de groupe (GPO)
  • Déploiement des profils itinérants
  • Configuration des partages réseau et des permissions
  • Automatisation des tâches avec PowerShell

Mise en place d'un outil de ticketing GLPI sous Linux

Mars 2023 Gestion de parc

Description du projet

Implémentation d'une solution complète de ticketing et de gestion de parc informatique basée sur GLPI (Gestionnaire Libre de Parc Informatique) sous Linux CentOS. Le système permet de gérer les incidents, les demandes et l'inventaire du matériel.

Fonctionnalités implémentées

  • Installation et configuration de GLPI avec MySQL
  • Intégration avec FusionInventory pour la découverte réseau
  • Configuration des workflows de tickets
  • Mise en place des notifications par email
  • Création de tableaux de bord personnalisés
  • Automatisation des sauvegardes

Application météo

Mai 2023 Développement mobile

Description du projet

Application mobile développée avec Flutter permettant d'afficher les prévisions météorologiques en temps réel pour la localisation actuelle ou pour des villes recherchées. L'application intègre des données provenant d'une API météo et offre des fonctionnalités avancées de visualisation.

Fonctionnalités implémentées

  • Géolocalisation pour obtenir la météo locale
  • Recherche par ville avec autocomplétion
  • Affichage des prévisions sur 5 jours
  • Graphiques des températures et précipitations
  • Mode sombre/nuit
  • Notifications pour les alertes météo
Flutter Dart API REST Firebase Google Maps
