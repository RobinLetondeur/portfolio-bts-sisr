Création d'un Active Directory sous Windows Server 2022

Janvier 2023 Infrastructure réseau

Description du projet

Ce projet consistait à mettre en place un Active Directory complet sous Windows Server 2022 pour une entreprise fictive de 50 employés. L'objectif était de centraliser la gestion des utilisateurs, des ordinateurs et des politiques de sécurité.

Fonctionnalités implémentées

  • Création d'un domaine Active Directory avec DNS intégré
  • Configuration des Unités Organisationnelles (OU) par services
  • Mise en place des stratégies de groupe (GPO)
  • Déploiement des profils itinérants
  • Configuration des partages réseau et des permissions
  • Automatisation des tâches avec PowerShell
